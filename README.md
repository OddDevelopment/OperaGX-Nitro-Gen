## How to use

1. **Install Python**
   - Make sure you have Python installed on your system.

2. **Open Console**
   - Open your command-line interface or terminal.

3. **Install Packages**
   - Run the following command to install the required packages:
     ```bash
     pip install requests
     pip install PyQt5
     ```

4. **Run the Script**
